Sur le vrai client Discord, l’API impose : tout le bloc embed (titre, texte…) d’abord, puis tous les boutons en dessous. On ne peut pas intercaler un bouton au milieu du texte.
Les pubs / captures type « Lusive » sont souvent : maquette, image unique (bannière + faux boutons dessinés), ou autre app — pas un message Discord standard. Notre aperçu HTML imite le design ; le bot envoie du texte + boutons comme Discord l’autorise.
