Fiches paiement — labo

Construis l’embed comme sur Discord, puis copie le JSON pour /paiement definir ou +paiement set.

Comment ça marche dans le bot ?
  • Les fiches sont stockées en MongoDB par clé : paypal, litecoin, ethereum, solana, ou n’importe quel nom (ex. usdt).
  • Sur Discord : +paypal, +litecoin, +eth, +sol affichent l’embed enregistré pour cette clé.
  • /paiement afficher (autocomplete) — prévisualiser · /paiement definir avec un JSON — enregistrer.
  • Préfixe : +paiement set <clé> <json> (message sur une ligne ou JSON compact).
  • Couleur dans le JSON : nombre décimal ou chaîne "#5865F2" (le bot convertit).
