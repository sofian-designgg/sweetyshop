Image → lien pour embed

Une page HTML ne peut pas « héberger » ton fichier sur Internet toute seule. Ici : envoi vers ImgBB (gratuit, clé API) pour obtenir un https://… utilisable dans +ticket panel-image, paiements-lab, etc.

1. Choisir une image

Glisse-dépose une image ici
ou clique pour parcourir
PNG, JPG, GIF, WebP — max ~32 Mo (limite ImgBB)

2. Clé API ImgBB

Crée une clé gratuite sur api.imgbb.com (inscription rapide). Colle-la ci-dessous — elle est mémorisée localement dans ton navigateur.

Si l’envoi échoue en ouvrant ce fichier en file://, lance un petit serveur : npx serve preview puis ouvre la page en http://localhost:3000/image-upload-lab.html (certaines protections du navigateur bloquent les appels API depuis un fichier local).

Sans ImgBB : lien Discord (CDN)

  1. Envoie ton image dans un salon Discord (serveur où le bot ou toi pouvez voir le média).
  2. Clic droit sur l’image → Copier le lien (ou « Copier l’adresse du média »).
  3. Tu obtiens une URL du type https://cdn.discordapp.com/attachments/… — colle-la dans ton embed.

Ce lien reste valide tant que le message existe (ne pas supprimer le message si tu veux garder l’image).
